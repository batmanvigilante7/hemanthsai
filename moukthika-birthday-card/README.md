# Moukthika Birthday Card 🌸

A single-file cinematic birthday experience inspired by the reference flow:

1. Developer-style birthday protocol intro
2. Fade from code into a dark cinematic scene
3. Growing tree and heart-shaped petal canopy
4. Personal message reveal
5. Tap/click to replay

## Run locally

No installation needed.

- Open `index.html` directly in a browser, or
- Use VS Code Live Server for a clean full-screen preview.

## Edit the message

Open `index.html` and update the copy inside:

```html
<article class="message" id="message">
```

## Capture as a reel

1. Open in Chrome.
2. Press `F11` for fullscreen.
3. Set the browser window to a 9:16 size in DevTools device mode.
4. Screen-record from the start.
